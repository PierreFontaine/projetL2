## Compilation

### pré requis

Vous êtes sous UNIX (Linux ou Mac), et disposez de Make.

### Make fait le taff pour vous !

Télechargez le dossier et accedez y avec le terminal !

```sh
  cd pathToDir/game/
  make
  ./programme
```
